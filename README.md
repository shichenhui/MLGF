# MLGF

The implement of "Multi-view clustering based on the association of graph structure and feature distribution".

![MLGF Figure2](https://github.com/shichenhui/MLGF/blob/main/Figure2.png)

## CODE Instruction

```
data_loader1.py: Data Loader	Data reading, preprocessing and batch loading
model58.py:	Model Definition	Neural network model architecture implementation
run6.py:	Main Script	Program entry point, training pipeline control
utils5.py:	Utility Functions	Various helper functions and utilities
evaluation.py:	Evaluation Module	Model performance evaluation
metrics.py:	Metrics Calculation	Performance metrics definition and computation
```

## RUN

The training of the model can be running by:

```
python run6.py
```
